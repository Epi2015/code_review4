Description:

Web application that takes an online Pizza order and returns a total cost:
User can choose multiple toppings/pizza size and include a delivery fee based on distance travelled.




SETUP:

Clone this repository. Open a browser to view in HTTP.
Technologies used:
Javascript
CSS
Bootstrap
HTML
